ITD Aggregate Materials Database

This tool is used to enter and review records in the ITD aggregate source materials database (RP-315).

Browse — View all sites, their records, test results, and QAMS qualifications. Click any site in the list to see its details. Click a Rec to see linked tests and QAMS entries.

Data Entry — Create new Rec (document) entries linked to a site. Optionally attach QAMS qualifications or test results in the same batch. Records are staged in a queue, then committed together.

+ New Site — Register a new material source site before entering records against it.

The server must be running on Christopher's workstation for this interface to function. If you see "OFFLINE" in the status bar, contact Christopher.
